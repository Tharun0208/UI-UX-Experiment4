# UI-UX-Experiment4

## Aim:
 To Design a wireframe diagram for a generic application or website using Balsamiq, including at least 5 essential screens/pages
 
## Algorithm:
1.Start

2.Open Balsamiq Mockups tool.

3.Create a new project for the application/website.

4.Design the following screens sequentially:

4.1.Welcome / Splash Screen: Add logo, app name, and Get Started button.

4.2.Login / Signup Screen: Add input fields, buttons, and authentication options.

4.3.Home / Dashboard Screen: Add navigation bar, cards, and search bar.

4.4.Content / Product Listing Screen: Add product grid, filters, and Add to Cart buttons.

4.5.Checkout / Contact Screen: Add form fields, total price, and Submit/Place Order button.

6.Connect the screens using navigation arrows to show flow:
Welcome → Login → Home → Product Listing → Checkout

7.Review layout consistency and usability.

8.Save & Export the final wireframe.

9.Stop
